# 2grupoSisA-Frigorifeco
 2grupoSisA - Frigorifico
Este projeto consiste em um sistema de gerenciamento para frigoríficos, desenvolvido pelo Grupo A da disciplina de Sistemas de Informação (SisA). O objetivo é controlar o fluxo de entrada, processamento e saída de produtos cárneos, garantindo a rastreabilidade e o controle de estoque.
 Funcionalidades
Gestão de Estoque: Controle de entrada de matéria-prima e saída de produtos acabados.
Controle de Temperatura: Monitoramento dos níveis de refrigeração das câmaras frias.
Cadastro de Fornecedores/Clientes: Gerenciamento de parceiros comerciais.
Relatórios: Geração de balanços de produção e vendas.
