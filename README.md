# 2grupoSisA - Frigorífico

Este projeto consiste em um sistema de gerenciamento de umidade e temperatura para frigoríficos, desenvolvido pelo **Grupo A** do curso de **Sistemas de Informação (Sis)**. O objetivo é observar, gerenciar e climatizar de forma correta as camâras frigoríficas.

## Funcionalidades

*   **Controle de Temperatura:** Monitoramento dos níveis de refrigeração das câmaras frias.
*   **Cadastro de Fornecedores/Clientes:** Gerenciamento de parceiros comerciais.
*   **Relatórios:** Dashboards e relatórios para melhor gestão e tomada de decisões efetivas.

## Tecnologias

*   **Linguagem:** C++, HTML, CSS, JS
*   **Banco de Dados:** MySQL
*   **Frameworks:** Ainda nenhum

## Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:
*   [Git](https://git-scm.com)
*   [Node.js](https://nodejs.org) ou [Python](https://www.python.org)

## Instalação e Execução

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/yurigarciapardinho/2grupoSisA-Frigorifico
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd 2grupoSisA-Frigorifico
    ```

3.  **Instale as dependências:**
    ```bash
    # Caso seja Node.js:
    npm install

    # Caso seja Python:
    pip install -r requirements.txt
    ```

4.  **Execute a aplicação:**
    ```bash
    # Caso seja Node.js:
    npm start

    # Caso seja Python:
    python main.py
    ```
